# 🐍 Snake Game

A simple **Snake Game** built using **HTML, CSS, and JavaScript** as part of my web development learning journey.

This project helped me practice **JavaScript game logic, DOM manipulation, and keyboard event handling** while recreating the classic snake game.

---
## 🎮 Demo

You can play the game here:
🔗 Live Demo: snake-game-mpwwybl3t-nabeel-builds-projects.vercel.app

## 🚀 Features

* 🐍 Snake movement using keyboard controls
* 🍎 Food generation for increasing score
* 📊 Score tracking system
* ❌ Game over detection
* 🎨 Simple and clean UI

---

## 🛠️ Tech Stack

* **HTML** – Structure of the game
* **CSS** – Styling and layout
* **JavaScript** – Game logic and interactions
